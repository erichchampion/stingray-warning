Stingray Warning is a cellular performance monitoring iOS application designed to track network connection quality and alert users about network conditions that may affect device performance. This privacy policy explains how we handle your information when you use our app.

üö´ Data We Do NOT Collect

Personal Information

  • Name, email address, or contact information
  • Phone numbers or device identifiers
  • Location data (beyond what's necessary for network analysis)
  • Browsing history or app usage patterns
  • Photos, videos, or other media
  • Financial information or payment data

üìä Data We Access (Local Analysis Only)

Cellular Network Information

Purpose: Performance monitoring and network quality analysis

Data Types:

  • Radio access technology (2G, 3G, 4G, 5G)
  • Carrier information (MCC/MNC codes)
  • Signal strength and quality metrics
  • Network change events and timing

Storage: All data is processed locally on your device and is not transmitted to external servers.

Location Data (Minimal)

Purpose: Providing context for network performance alerts

Usage: Location data is used only to provide geographical context for network performance alerts. This helps identify patterns in network quality across different areas.

Storage: Location data is not stored permanently and is not transmitted to external servers.

üì± Permissions Explained

Location Permission

How we use it: Only to correlate network events with location for better performance monitoring

Notifications Permission

Why we need it: To alert you when network performance issues are detected

How we use it: To send immediate alerts about network conditions that may affect performance

Your control: You can disable notifications in iOS Settings
